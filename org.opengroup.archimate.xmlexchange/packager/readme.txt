Packager Notes
--------------

1. Select the projects, "org.opengroup.archimate.xmlexchange" and "org.opengroup.archimate.xmlexchange.help"
2. From the context menu, select "Export..." and then "Deployable plug-ins and fragments"
3. Click "Next" and ensure the two projects are selected and output dir is "C:\"
4. Finish.
5. Run the packager.xml Ant script

The resulting zip file is the packaged plug-in.
